# DayCounter

iOS 날짜 계산기 애플리케이션입니다.

## 주요 기능
- **날짜 차이 계산**: 두 날짜 사이의 일 수를 계산합니다.
- **날짜 계산**: 기준 날짜에서 특정 일 수를 더하거나 뺀 날짜를 계산합니다.
- **한국어 지원**: UI 및 달력 포맷이 한국어에 최적화되어 있습니다.

## 기술 스택
- Swift
- SwiftUI

## 개발 내용
- `ContentView.swift`: 메인 UI 및 탭 뷰 구현
- `DateCalculator.swift`: 날짜 계산 로직 구현
- `DayCounterApp.swift`: 앱 진입점
- UI 및 캘린더 한국어 로컬라이제이션 적용
